Enhanced Shopping List is a simple web application to manage your shopping list, allowing you to add, remove, mark items as bought, and filter the list. Data is stored in LocalStorage, keeping the list even after closing the browser.

Features

Add items to the list.

Mark items as bought or pending.

Remove items individually.

Filter items: All, Pending, Bought.

Clear the entire list at once.

Dynamic counter (total, pending, bought).

Data persistence using LocalStorage.

Technologies

HTML5

CSS3

JavaScript (ES6)

LocalStorage

How to use

Open index.html in your browser.

Enter the item name and click Add or press Enter.

Use the filters to view pending, bought, or all items.

Click Clear All to remove the entire list.

Future Improvements

Edit existing items.

Drag and drop to reorder items.

Save the date of item addition.

Full responsive design.

Export/import list as CSV or JSON.
